# Plainrs

簡単な File Viewer & Writter


## Usage

```sh
Text Viewer & Writer 0.1.0

Hirotaka Kawashima

USAGE:
    plainrs [OPTIONS]

FLAGS:
    -h, --help       Print help information
    -V, --version    Print version information

OPTIONS:
    -r, --read <FILE>            
    -w, --write <FILE> <TEXT> 
```

### Read file

```sh
$ plainrs -r samples/text.txt
```


### Write file

```sh
$ plainrs -w samples/file.txt "Sample Text!"
```
